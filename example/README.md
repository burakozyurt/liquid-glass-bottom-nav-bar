# liquid_glass_bottom_navbar_plus example

A demo app showcasing every feature of
[`liquid_glass_bottom_navbar_plus`](https://pub.dev/packages/liquid_glass_bottom_navbar_plus),
with a light/dark toggle.

```sh
flutter run
```

## What's inside

| Page | Shows |
|------|-------|
| **Home** | The drop-in bar over scrolling content. |
| **Refraction** | Edge lensing close-up over a busy backdrop. |
| **Minimize behaviors** | The four `MinimizeBehavior` modes side by side. |
| **Search morph** | The search tab morphing the capsule into a text field. |
| **Accessory** | A mini-player shelf collapsing with the bar. |
| **Settings playground** | Every `LiquidGlassSettings` constant tuned live. |

> Run on a real device or an Impeller-enabled emulator to see the shader path;
> elsewhere the bar renders its frosted fallback.
