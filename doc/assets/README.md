# Demo assets

GIFs used by the main README. They are kept in the repository and referenced
from the README via absolute `raw.githubusercontent.com` URLs (pub.dev does not
resolve relative paths in HTML `<img>` tags, only in markdown images). They are
excluded from the published package archive via `.pubignore`.

| File | Shows |
|------|-------|
| `hero.gif` | Top-of-README hero — edge refraction / lensing over vivid content. |
| `minimize.gif` | Scroll-to-minimize collapsing the capsule. |
| `selection.gif` | Spring-driven selection pill. |
| `navigation.gif` | Page transitions across destinations. |
| `playground.gif` | The settings playground tuning the glass live. |

Source clips were ~960×2082 screen recordings; re-encoded to ~280–340 px wide,
10–15 fps, palette-optimized (≈1–2 MB each) with `ffmpeg`.
